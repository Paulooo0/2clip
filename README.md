# 2clip

2clip is a simple CLI tool for managing your clipboard

### Installation

#### Unix

Run the following command, to make the script executable:
```bash
chmod +x install_2clip.sh
```

Run the bash script:
```bash
./install_2clip.sh
```

#### Windows

Open PowerShell as an `administrator`.

Run the PowerShell script:
```shell
.\install_2clip.ps1
```
